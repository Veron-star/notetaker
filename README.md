## Description:
Create an application that can be used to write, save, and delete notes. This application will use an express backend and save and retrieve note data from a JSON file. 
    
## Command Used:
- NPM Install
- NPM Install Inquirer
- JSON
- NodeJS
- Express
- Heroku
